Installation in 4 Steps

  1. Upload the basicseo folder to /wp-content/plugins/
  2. Activate the plugin through the Plugins menu in WordPress
  3. Navigate to BasicSEO in your admin menu
  4. Configure settings and generate your files

Frequently Asked Questions

Does BasicSeo automatically update my sitemap?

No. BasicSeo is fully manual. You control when sitemaps are generated or updated.

Can I use this alongside other SEO plugins?

Yes. BasicSeo detects potential conflicts and alerts you. It is designed to respect existing files.

What if my robots.txt or ads.txt directory is not writable?

The plugin provides a virtual fallback so your rules still apply even if physical file creation fails.
