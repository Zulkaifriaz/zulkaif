BasicSeo

A lightweight, manual SEO file manager for WordPress. Generate and control sitemaps, robots.txt, ads.txt, and other essential SEO files with **full transparency** and no silent automation.

View Source on GitHub

Key Features

Manual Sitemap Generation

Create XML sitemaps with full control over included content and preview before going live.

Robots.txt Management

Edit and preview robots.txt safely without overriding WordPress core.

Ads.txt Editor

Manage advertising declarations via entry-based or raw editing with validation.

Conflict Detection

Alerts you to potential conflicts with other SEO plugins to avoid issues.

No Silent Automation

All actions require explicit user approval. No changes are made without your consent.

WordPress Native

Built using WordPress APIs and coding standards, respecting existing plugins and files.

Installation

  1. Upload the basicseo folder to /wp-content/plugins/
  2. Activate the plugin through the 'Plugins' menu in WordPress
  3. Navigate to BasicSeo in your WordPress admin menu
  4. Configure settings and generate your sitemap
